# Language Flashcards App

Add card sets and cards, with data stored in the Vercel-PostgreSQL database connected to a REST API.

## Used Technologies

- vercel serverless postgresql
- tailwindcss
- shadcn ui
- api routes
- react query (for server state management)
- axios
- zustand (for client state management)
