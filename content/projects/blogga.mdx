## Blogga: Next.js Contentlayer Mdx Blog

Blogga is a dynamic and interactive blog site meticulously crafted with cutting-edge technologies, leveraging the powerful combination of ContentLayer MDX and Next.js framework. Seamlessly blending the simplicity of Markdown syntax with the flexibility of React components.

![Alt Text](https://i.ibb.co/R3D67bH/Group-1-1.png)

Used technologies:

- Framework: Next.js
- Styling: scss
- Deployment: netlify
- MDX: contentlayer
